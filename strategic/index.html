a reading · in six lenses

Read it one element at a time.

A decision, a situation, a strategy — anything you're trying to read clearly. Each element is one lens. You start with the one that fits what you're carrying, walk three choices, and it cuts a single line — a nemetic.φ — that you can take deeper with the guide, then hand on to the next element. Six lenses, and a synthesis at the end.

Start anywhere. Begin with the lens that fits — the reading rotates through all six from there.
